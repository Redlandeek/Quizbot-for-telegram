# Telegram Quiz Bot

Бот для проведения квизов по программированию с сохранением статистики игроков.

## Команды бота

### Основные команды

- `/start` - Запустить бота и показать главное меню
- `/quiz` - Начать новый квиз
- `Начать игру` - Начать новый квиз (кнопка)

### Статистика

- `/stats` - Показать вашу последнюю статистику
- `Моя статистика` - Показать вашу статистику (кнопка)
- `/leaderboard` - Показать таблицу лидеров
- `Таблица лидеров` - Показать таблицу лидеров (кнопка)

## Функционал

### Квиз
- 10 вопросов по программированию
- Интерактивные кнопки для ответов
- Автоматическое удаление кнопок после ответа
- Подсчет правильных ответов в реальном времени

### Статистика
- Сохранение последнего результата каждого пользователя
- Отображение процента правильных ответов
- Таблица лидеров с топ-10 игроков
- Дата и время прохождения квиза

### Особенности
- Удобное главное меню с кнопками
- Чистый интерфейс без лишних элементов
- База данных для хранения прогресса и результатов
- Модульная структура кода
